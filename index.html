Delayappstart

Intended for use in the "startup" programs group (XP/7), choose list of apps to start on logon and the delay time between each. HTML/JavaScript.

October 8th 2014

In its current state this program has some statically entered application paths that each execute after a specific amount of time in order.

The next stage will include being able to enter in custom paths and specifying some number of seconds between 1 and 59.

The reason for this programs existence was very simple: one of the PCs I use at work is very slow when I first logon and it takes a while for 5 or 6 applications to show up on screen as I click each one off on the task bar. This program simply executes a program, waits some number of seconds then executes the next one, and so on. Very basic functionality that could accomplished with a CMD batch file. I decided to write this as an HTA for multiple reasons including but not limited to: I can use this HTA as a framework for other utilities with similar requirements and I needed practice with HTML/JavaScript anyway.

The program needs at least two modes: the first one is for adding and removing applications from the list, the "editor mode if you will. The second mode is where upon loading the program starts executing without user interaction.

Vision for the "final" version: Specify a path and time out time for any number of applications save changes to a config file (probably XML) Also automatic placement of a copy of program to the startup group of the current user's profile (and an option to put it in the "all users" start up group although I can't think of a reason why you would want to do that).

Right now I am leaning toward writing out a customized version of the DelayAppStart rather than creating a custom shortcut or something like that because a shortcut can be so easily broken simply by moving the original HTA file.
